# EduTrack

A web application for managing assignments, grades, and courses 
between students and instructors.

This is our case study project for Python Web Framework subject.

---

## About the Project

We built EduTrack as a full-stack project. Students can submit 
assignments, check their grades, and see announcements. Instructors 
can manage courses, grade submissions, and post updates.

---

## What We Used

- Backend: Django + Django REST Framework
- Database: SQLite3
- Frontend: AngularJS
- Authentication: JWT tokens

---

## Features

1. Assignment Submission
2. Student Progress Dashboard
3. Instructor Feedback and Grading
4. Course and Module Management
5. Announcement Board
6. Grade Summary
7. Deadline Tracker
8. Submission History

---

## How to Run

**Backend:**

   - cd backend
   - venv\Scripts\activate
   - python manage.py runserver

**Frontend:**
Open index.html with Live Server in VS Code
